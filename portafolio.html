Hospital Plus

Hospital Plus es un sistema desarrollado en C# con .NET Framework. Utiliza SQL Server Management como base de datos, y cuenta con funcionalidades como:

  • Inicio de sesión con roles
  • Gestión de usuarios, médicos y pacientes
  • Consultas, citas, historial médico y recetas
  • Generación de reportes con Crystal Reports
  • Integración completa con base de datos en SQL

NeoTech

NeoTech es una tienda en línea desarrollada con HTML, CSS, JavaScript. Utiliza BootsTrap v5.3 como Framework para creación del diseño.El Sitio Web tiene funcionalidades como:

  • Inicio y Registro de sesión
  • Inicio, Nosotros, Contacto, Noticias
  • Tienda con buscador y menu vertical para filtrar
  • Generación de formularios de contacto
  • Carrito funcional para proceder compra

MyBook

MyBook es un sistema de biblioteca desarrollado en C# con .NET Framework. Utiliza SQL Server Management como base de datos, y cuenta con funcionalidades como:

  • Inicio de sesión con roles - Admin y Usuarios
  • Gestión de usuarios, bibliotecario y lectores
  • Reservar, prestamos, devoluciones historiales de usuarios, préstamos y devoluciones
  • Generación de informes de historiales
  • Integración completa con base de datos en SQL
